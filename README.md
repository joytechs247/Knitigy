# Legal Services Platform

A professional, modern, and responsive marketing site for legal/consultancy services built with Next.js, Tailwind CSS, and Framer Motion.

## Features

- 🏠 Professional legal services platform
- 📝 Blog system with markdown content
- ✨ Smooth animations with Framer Motion
- 📧 Contact forms with EmailJS integration
- 🎨 Tailwind CSS for styling
- 📱 Fully responsive design
- ♿ Accessibility focused
- 🔍 SEO optimized

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Forms**: EmailJS
- **Content**: Markdown files
